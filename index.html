Welcome to Vagrant

What is Vagrant?

Vagrant is a tool to Create and configure lightweight, reproducible, and portable development environments.

Why Vagrant?

  • No more "runs on my machine"
  • Same enviroment as in production
  • Portable and runs on every OS
  • Easy to set up, also for non tech people
  • Runs on Virtualbox, AWS, Docker...

Getting started

  1. Install Vagrant
  2. Install Virtualbox
  3. Install NFS plugin for Windows
    4. 							
								$ vagrant plugin install vagrant-winnfsd

Getting Started

					  	
					  		$ mkdir vagrant_getting_started
   						
   						
					  		$ cd vagrant_getting_started
   						
   						
					  		$ vagrant init
